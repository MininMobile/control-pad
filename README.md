# Control Pad
Macros for my numpad using AHK and Lua Macros.

## Requirements
- [AutoHotKey](https://autohotkey.com/) (The One with Unicode)
- [Lua Macros](http://www.hidmacros.eu/forum/viewtopic.php?t=241)
